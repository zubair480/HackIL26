## Development Workflow

```
1. Change code in models/, routes/, or decorators/
2. Flask auto-reloads (debug=True in development)
3. Test with: python test_api.py
4. Commit when working

For production:
1. Set env variables
2. gunicorn wsgi:app
3. Put behind Nginx
4. Done!
```
